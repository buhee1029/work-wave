## 프로젝트 개요
> 본 서비스는 기업 내부에서 사용되는 웹 서비스로, 프로젝트와 작업 관리를 위한 직관적이고 유연한 도구로, 팀 간 협업과 업무 효율성을 향상시키는데 목적을 두고 있습니다.

<br/>

## 개발환경
```
• IDE : IntelliJ IDEA Ultimate
• 언어 : Java 17
• 프레임워크 : Spring Boot 3.1.5
• 빌드 도구 : Gradle
• 데이터베이스 : MySQL 8.0
```
